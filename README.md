# Parrot

This bot is a rewrite of the original Parrot bot. Please refer to the original repository for more information: https://github.com/rtk-rnjn/Parrot

#### Features?

The bot will not have any features from the original Parrot bot. The rewrite is feature-on-demand. If you want a feature contact the maintainer/original author, open an github issue or just DM the bot or the owner.
